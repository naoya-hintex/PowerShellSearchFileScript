# Powershellファイル検索スクリプト

## 📌 概要
指定フォルダ内からファイル名・パス・更新日をまとめて取得する Powershell スクリプトです。

---

## 🚀 特徴 / 機能
- 特徴1  指定フォルダ内にあるサブフォルダに対しても検索を実行できます。
- 特徴2  エクスプローラーの検索機能やVBAでマクロを作って検索するより、素早く検索ができます。
- 特徴3  検索結果をCSVファイルにエクスポートします。
- 特徴4  コード量が短く、パラメータの設定やメンテナンスが簡単です。

---

## 🛠 使用技術
- 言語 / フレームワーク PowerShell  
- 実行環境             Windows 10 以降
